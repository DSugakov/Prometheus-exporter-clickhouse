# Структура ВКР (черновик)

1. Введение: актуальность мониторинга ClickHouse, ограничение доступа к server-side конфигу.
2. Аналитический раздел: архитектура ClickHouse, источники метрик, обзор аналогов (в т.ч. community exporter), классификация метрик.
3. Постановка задачи и требования: внешний экспортёр, профили сборов, безопасность, совместимость версий.
4. Проектирование: модули коллекторов, конфигурация, ограничение нагрузки и cardinality.
5. Реализация: язык Go, ключевые компоненты, сценарии развёртывания.
6. Эксперименты: baseline, методика подсчёта метрик, стенд, ограничения, сравнение с эталоном.
7. Заключение и направления развития.

Приложения: конфиги, фрагменты кода, примеры метрик, Grafana dashboard.
